# Personal Portfolio Website

## 📌 Project Overview

This project is a Personal Portfolio Website developed using HTML, CSS, and JavaScript. The website serves as a digital portfolio to showcase personal information, skills, projects, achievements, and contact details in a visually appealing and responsive format.

## 🚀 Features

* Responsive and Modern Design
* Home Section with Introduction
* About Me Section
* Skills Showcase
* Projects Portfolio
* Education and Experience Details
* Contact Information Section
* Smooth Navigation and User Experience

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
